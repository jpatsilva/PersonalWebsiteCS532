Secure Key Generation

Background

I developed a secure key generation application. The key generation application follows a set of well defined requirements which includes specific formats and algorithm used to create a payload. The payload is used by a classified piece of equipment to pack and unpack (encrypt and decrypt) the secure key. The secure key is used to enable and disable equipment function.

Tools Used to Implement
Integrated Development Environment (IDE) Microsoft Visual Studio 2019
